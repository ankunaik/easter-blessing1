## How to Use

1. **Add Your Image**
   - Place your high-quality risen Jesus Christ image in the `images/` folder
   - Name it `risen-jesus.jpg` (or change the filename in `index.html` line 27)
   - For best quality, use a high-resolution image (1200px or higher width)

2. **Open in Browser**
   - Open `index.html` in any web browser
   - Click on the "Happy Easter Blessing" link to view the Easter greeting

3. **Customize**
   - Edit the greeting text in `index.html` (lines 26-27)
   - Modify colors in `css/styles.css`
   - Adjust animation speeds in `css/styles.css` or `js/script.js`

## Browser Support

Works in all modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers

## Customization Tips

### Change Colors
- Edit the gradient colors in `css/styles.css`
- Look for lines with `linear-gradient()` values

### Adjust Animation Speed
- Find the `@keyframes float` animation in `css/styles.css`
- Modify the duration values in the cloth elements (e.g., `5s`, `6s`, `7s`)

### Change Text
- Edit `index.html` to change:
  - Welcome message (line 10)
  - Easter greeting (line 26)
  - Biblical verse (line 39)

## Tips for Image Quality

1. Use Ultra HD images (1920x1080 or higher)
2. Ensure good lighting in the photo
3. Remove backgrounds if needed for better visual effect
4. Optimize the image size while maintaining quality (compress if > 2MB)
